# Grocery Shop

**Автор:** Тулегенова Карина

## Отчет по проекту

`Grocery Shop` - мобильное приложение интернет-магазина продуктов, разработанное на Flutter. Проект демонстрирует базовый пользовательский сценарий: знакомство с приложением, просмотр каталога, открытие карточки товара, добавление продуктов в корзину, удаление товаров и расчет итоговой стоимости покупки.

Приложение ориентировано на простой и понятный интерфейс: пользователь сразу видит список продуктов, может быстро перейти в корзину или профиль, а навигация между основными разделами выполнена через нижнее меню.

## Цель проекта

Целью проекта было создать кроссплатформенное Flutter-приложение с несколькими экранами, локальным состоянием корзины и удобной навигацией. В ходе работы были закреплены навыки построения UI, подключения изображений, работы с состоянием через `Provider` и организации маршрутов через `go_router`.

## Основные возможности

- Стартовый экран с логотипом и кнопкой перехода в магазин.
- Главная страница с каталогом продуктов.
- Карточки товаров с названием, изображением, ценой и кнопками действий.
- Экран подробной информации о выбранном товаре.
- Корзина покупок с добавленными товарами.
- Удаление товаров из корзины.
- Автоматический расчет общей суммы заказа.
- Профиль пользователя с основной информацией.
- Нижняя навигационная панель для быстрого перехода между разделами.
- Обработка неизвестных маршрутов через отдельный экран ошибки.

## Использованные технологии

- **Flutter** - основной фреймворк для разработки приложения.
- **Dart** - язык программирования проекта.
- **Provider** - управление состоянием корзины.
- **go_router** - декларативная навигация между экранами.

## Архитектура и логика работы

Приложение запускается из файла `main.dart`. В корневом виджете подключается `ChangeNotifierProvider`, который передает экземпляр `CartModel` всем экранам приложения. Благодаря этому корзина остается доступной из каталога и со страницы корзины.

Модель `CartModel` хранит список доступных продуктов и отдельный список товаров, добавленных в корзину. В модели реализованы методы добавления, удаления и подсчета итоговой суммы. После каждого изменения вызывается `notifyListeners()`, поэтому интерфейс автоматически обновляется.

Навигация вынесена в `app_router.dart`. Для переходов используются маршруты:

- `/` - стартовый экран;
- `/home` - главный экран с товарами;
- `/details/:id` - страница деталей товара;
- `/cart` - корзина;
- `/profile` - профиль пользователя.

## Пользовательский сценарий

1. Пользователь открывает приложение и попадает на стартовый экран.
2. Нажимает кнопку `Get started` и переходит в каталог.
3. Просматривает продукты и добавляет нужные товары в корзину.
4. При необходимости открывает подробную информацию о товаре.
5. Переходит в корзину, проверяет список покупок и итоговую сумму.
6. Может удалить лишние товары из корзины.
7. Через нижнее меню переходит между главной страницей, корзиной и профилем.
